Introducing UliCMS Portable

UliCMS 8.0.1 Portable for Windows is an instance of XAMPP that comes preinstalled with UliCMS 8.0.1.

It makes it very easy, to try out UliCMS without the need to setup a webserver.

Other versions of UliCMS and versions for other operating systems will follow.

Please note:

Don't use this software for hosting public websites, since it have an unsecure configuration by default.

It's only a test and development environment.

Howto install

Simple extract the "xampp" folder from zip file and copy it into the root directory of your harddrive C:\

Howto run

  1. Open c:\xampp and run "xampp-control" application.
  2. Select your language
  3. Click on "Start" next to "Apache". You maybe have to confirm warnings of your computer firewall.
  4. Click on "Start" next to "MySQL". You maybe have to confirm warnings of your computer firewall.
  5. Open your browser and enter "http://localhost/admin" into the address bar of your web browser.
  6. Login.
    Default login is:
    User: admin
    Password: password
    it's recommend to change this immediately.
  7. When you finished using UliCMS then stop the "Apache" and "MySQL" modules using "XAMPP Control Panel" and quit the application.
