Jetzt neu: UliCMS Portable

UliCMS 8.0.1 Portable für Windows ist eine Instanz von XAMPP die mit einem vorinstallierten UliCMS 8.0.1 kommt.

Diese Software macht es sehr einfach, UliCMS auszuprobieren, ohne sich mit der Einrichtung und Konfiguration eines Webservers rumplagen zu müssen.

Andere Versionen von UliCMS und Versionen für andere Betriebssysteme werden demnächst erscheinen.

Achtung:

Nutzen Sie diese Software nicht, um öffentliche Internetseiten zu hosten, da sie unsicher konfiguriert ist.
Diese Software dient nur als Test- und Entwicklungsumgebung.

Installationsanleitung

Entpacken Sie einfach den Ordner "xampp" und kopieren Sie diesen in das Hauptverzeichnis eine Laufwerks (z.B. nach C:\).

Starten, Nutzung und Stoppen

  1. Öffnen Sie den Ordner c:\xampp und starten Sie das Programm "xampp-control".
  2. Wählen Sie Ihre Sprache aus.
  3. Klicken Sie beim Modul "Apache" auf "Starten". Eventuell müssen Sie Sicherheitsnachfragen von Ihrer Computer Firewall bestätigen.
  4. Klicken Sie beim Modul "MySQL" auf "Starten". Eventuell müssen Sie Sicherheitsnachfragen von Ihrer Computer Firewall bestätigen.
  5. <Öffnen Sie Ihren Webbrowser und öffnen Sie die Adresse: http://localhost/admin
  6. Melden Sie sich an.
    Die Standard-Zugangsdaten sind:
    Name: admin
    Passwort: password
    Es wird empfohlen, das umgehend zu ändern..
  7. Wenn Sie fertig mit der Nutzung von UliCMS sind, "Stoppen" Sie die Module "Apache" and "MySQL" im "XAMPP Control Panel" und beenden Sie das Programm.
