Patch Management

Ab UliCMS 8.0.2 können Patches automatisch heruntergeladen und installiert werden.

Damit diese Funktionalität genutzt werden kann, müssen die Dateirechte so frei gesetzt sein, dass PHP auf sämtliche Dateien der Installation Lese- und Schreibzugriff hat und PHP
Daten von externen Servern nachladen kann (per fsockopen oder curl).

Patches installieren

Sofern Patches verfügbar sind, und Ihr Benutzer-Account das Recht system_update hat,
werden Sie in dem "Willkommen" Bereich im Backend darüber informiert.

Achtung:
Auch wenn die Patches sorgfältig getestet werden, können Fehler nie ausgeschlossen werden.
Wir empfehlen Ihnen deshalb, zuvor eine Sicherung des Systems durchzuführen.

Klicken Sie auf "Patches installieren". Sie bekommen nun eine Liste der Verfügbaren Patches angezeigt.

Wählen Sie die zu installierenden Patches aus, in dem Sie die Häckchen setzen und klicken Sie auf
"ausgewählte Patches installieren". Anschließend werden alle ausgewählten Patches heruntergeladen und entpackt.
