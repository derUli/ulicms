UliCMS 2020.2 with CKEditor updates and visual revised

UliCMS 2020.2 bietet Ihnen eine Vielzahl neuer Features zur Gestaltung Ihrer Webseite. Dieses Release ist sowohl für neue Anwender als auch als Upgrade für bereits bestehende Projekte geeignet.

Visual Revised Backend

Das UliCMS Backend wurde optisch überarbeitet. So wurden die Abstände reduziert und die verfügbare Anzeigefläche auf dem Bildschirm wird nun besser genutzt.

Die Benutzer die gerade online sind werden nun mit Avatar angezeigt.

Dropdown Filters

Die Dropdownfilter in der Seitenverwaltung sind nun wieder hinzugefügt. Allerdings konnte dieses Feature bis zum Release Datum nicht ausreichend getestet werden. Daher ist das Feature in der ausgelieferten Version deaktiviert.

Wenn Sie das Filter Feature testen möchten, ist dafür ein Patch erforderlich. Wenn Sie diesen Patch erhalten möchten, kontaktieren Sie mich bitte.

CKEditor Updates

Die CKEditor Komponente, welche das Formatieren von Inhalten, ohne HTML-Kenntnise ermöglicht, wurde auf Version 4.13.1 aktualisiert.

Alle CKEditor Instanzen merken sich jetzt die Größe wenn Sie die Eingabefelder vergrößern oder verkleinern.

More new Features

Benutzer können nun ihr Profilbild entfernen. In diesem Release wurden Bugs und Sicherheitsprobleme in UliCMS behoben.

Das Changelog kann nun in der core_info Komponente eingesehen werden.
