UliCMS 2020.2 with CKEditor updates and visual revised

UliCMS 2020.2 includes a lot of new features and improvements. This release is suited for new users and as upgrade for existing projects.

Visual Revised Backend

The UliCMS Backend is visual revised. The margins were reduced and the user interface now uses the available screen more efficient.

The current online users are now shown including their avatar pictures

Dropdown Filters

While the dropdown search filters were reimplemented, the feature is disabled in this release, because it weren't enough tested when reaching the release deadline.

If you are interested in testing this feature you can request a patch that enables this feature. Just contact the developer.

CKEditor Updates

The CKEditor components, which enables users to format contents without the need to have html knowledge, was updated to version 4.13.1.

All CKEditor instances do remember and restore their size after they were resized.

More new Features

Users can remove their profile avatar image. In diesem Release wurden Bugs und Sicherheitsprobleme in UliCMS behoben.

The release changelog was added to the core_info component.
