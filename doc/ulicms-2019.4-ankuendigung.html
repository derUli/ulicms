UliCMS 2019.4 mit automatischer Einbindung von Medien, hierarchischer Seitenliste, einem neuen Standard-Theme und vielen weiteren neuen Features

UliCMS 2019.4 bietet Ihnen eine Vielzahl neuer Features zur Gestaltung Ihrer Webseite.

Support für ältere MariaDB Versionen

Die vorherige UliCMS Version 2019.3 ließ sich nur ab MariaDB 5.6 bzw einer vergleichbaren MySQL Version installieren. UliCMS 2019.4 ist nun wieder kompatibel mit MariaDB Version ab Version 5.5.3.

Automatische Einbindung von externen Medien

URLs von externen Medien werden bei Aufruf einer Seite automatisch durch eingebettete Media Player ersetzt. Es werden über 150 Dienste unterstützt. Unter den unterstützten Diensten befinden sich unter anderem die Folgenden:

  • YouTube
  • Dailymotion
  • MyVideo
  • Vimeo
  • Ustream

Hierarchischer Seitenverwaltung

Die Seitenliste ist nun hierarchisch aufgebaut, näheres dazu im unten verlinkten Dokument "Neue Features in UliCMS 2019.4".

Neues Theme "2020"

UliCMS 2019.4 enthält als zusätzliche Gestaltungsoption das neue Single-Page Theme 2020. Wenn Sie kommerzielle Projekte auf Basis dieses Themes umsetzen möchten, lesen Sie bitte den Hinweis zur Lizenzierung von fullPage.js.
