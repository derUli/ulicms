UliCMS 2018.1 is now available

UliCMS is the current major release of the german content-management-solution from germany. The developer released this new version to support users with new features and bugfixes.

UliCMS 2018.1 containts a wide-range of new features and bugfixes:

  • Parts of the backend were moved to a MVC pattern
  • Modules may create content types with additional typified fields
  • Enhanced API
