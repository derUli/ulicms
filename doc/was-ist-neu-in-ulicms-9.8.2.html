UliCMS 9.8.2 ist das neueste Update für Benutzer von UliCMS 9, so dass Sie weiterhin Ihre Webseiten damit betreiben können.
Dieses neue Release bietet eine Vielzahl neuer und verbesserter Funktionen.

Die größten Neuerungen

  • Seiten erstellen & bearbeiten nun mit Tabs
  • Verbesserte Systemsicherheit
  • Aktualisierter CKEditor (Version 4.5.6)
  • Aktualisierte Mobile_Detect Klasse (Version 2.8.19)
  • Erweiterungen der API
  • Kommandozeilen Script sinstall.php hinzugefügt

Die Upgrade Prozedur funktioniert wie immer. Ein Upgrade kann von UliCMS 9.0.0, 9.0.1, 9.8.0 oder 9.8.1 als Ausgangsbasis durchgeführt werden.
