UliCMS 9.8.5 is here

UliCMS 9.8.5 is the newest release of UliCMS 9. With UliCMS 9.8.5 you can continue to run your website. The new version offers many new features including:

  • A new menu based installation wizard
  • New package format SimpleInstall v2
  • Prepared Statements
  • Content Type "Article"
  • Cache Control
  • Many new and updated packages
  • Show/Hide headlines
