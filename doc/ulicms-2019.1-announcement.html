UliCMS 2019.1 has many new features

UliCMS 2019.1 is the first UliCMS release in 2019.
This release focuses on community features, antispam features and usability improvements.

Major Features in this Release

  • Comments
  • Additional spam prevention options
  • Assign multiple groups to users
  • "Delete" Button in the "edit page" view

Comments

Starting with this release UliCMS features an integrated and full featured comments solution. The comments feature is disabled by default but can be easily enabled. The only official Theme which currently supports comments is "impro17". However it's very easy to integrate the comments form into other themes.

The comments feature supports all options to prevent spam submissions of beeing released and an easy to use backend for managing comments.

Assign multiple groups to users

In addition to the primary group it's now possible to assign multiple secondary groups to a user. The permission arrays will be merged. If any of the group assigned to a user has a permission checked then the user owns that permissions.
