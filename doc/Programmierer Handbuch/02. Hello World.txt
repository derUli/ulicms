02 - Hallo Welt
________________________________

Die erste Anwendung die man wenn man eine neue Programmiersprache lernt programmiert ist eine Anwendung die nichts weiter macht als einen Text wie "Hallo Welt!" auf dem Bildschirm auszugeben.

1. Wechseln Sie in das UliCMS-Hauptverzeichnis.
   Gehen Sie in den Ordner "modules"
2. Legen Sie einen Ordner mit dem Namen "hello_world"
   an (ohne Anführungszeichen)
3. Im soeben erstellen Ordner legen Sie eine PHP-Skriptdatei an mit dem Namen "hello_world_main.php".
4. Öffnen Sie diese Skriptdatei fügen Sie einen PHP-Codeblock ein.
5. Erstellen Sie die Funktion hello_world_render() diese Funktion gibt einen String zurück (return), der String kann HTML-Code erhalten.


Listing Hello World:
 <?php
   function hello_world_render(){
    
    return "<p>Hallo Welt!</p>";
   }
 ?>

6. Kopieren Sie den Code zur Einbettung des hello_world Moduls in eine neue Seite und Speichern Sie diese
7. Rufen Sie die Seite im Frontend der Website auf.

Statt "hello_world" kann das Modul auch einen beliebigen anderen Namen haben.
Der Name eines Moduls kann jedoch nur Zeichen enthalten die sowohl in PHP-Funktionsnamen als auch in Dateinamen vorkommen dürfen.
Die render-Methode ist zwingend erforderlich, damit eine Erweiterung von UliCMS als solche erkannt wird. Auch wenn es sich um ein Modul handelt, welches keine Ausgaben im Frontend macht.
