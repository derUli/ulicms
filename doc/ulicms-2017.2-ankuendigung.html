UliCMS 2017.2 ist da!

UliCMS 2017.2 ist das UliCMS Release für das erste Quartal 2017. UliCMS 2017.2 ermöglicht Webmastern, weiterhin ihre Webseiten zu betreiben.

UliCMS 2017.2 enthält eine Vielzahl neuer Features und Verbesserungen. Dazu gehören:

  • Neues Backend Design
  • Paketmanagement und Updatesystem nutzt nun https://
  • Funktion ModuleHelper::getFirstPageWithModule() hinzugefügt
  • DBMigrator hinzugefügt

Hinweise zum Upgrade

Ein Upgrade kann nur von Version 2017.1 als Basis erfolgen. Bevor Sie ein Upgrade auf UliCMS 2017.2 durchführen, stellen Sie sicher, dass bei Ihrem Benutzer die Option "Ist Admin" aktiv ist.

Nach dem Upgrade müssen Sie die neuen Berechtigungen vergeben. Wenn Sie zuvor Mobile_Detect genutzt haben, müssen Sie das Modul Mobile_Detect aus UliCMS eXtend installieren. Außerdem sollten Sie mit update_manager auf Aktualisierungen prüfen und diese installieren.
