UliCMS 2018.4 has many new features

The new UliCMS Release 2018.4 is the result of 3 months of development.

The release 2018.4 focuses on usability, privacy and performance features.

Major Features in this Release

  • Usability Improvements
  • Support for the legal requirements of the General Data Protection Regulation (GDPR)
  • Better Performance

Usability Improvements

Tables are now shown with the DataTable jQuery Plugin. Tables now have a pagination. There is an option to configure how many entries are shown per page. As an extra there is now a live search for all datatables.

The forms "create page" and "edit page" are now clearer. The count of areas were reduced by joining areas with similiar topics and by moving rarely used options to the new "Others" accordion area.

Support for GDPR Requirements

The new privacy settings has an option to add a GDPR accept privacy checkbox. The checkbox may be integrated into forms and with an API to your own modules.

The Right to receive information about the stored data and the right to request the deletion of the stored data can be implemented with the module personal_data_export.

Improved Performance

The performance was improved. Indices was added to some database tables.

Javascript- and CSS codes are now better minified. Additionally obsolete spaces and newlines are removed from the HTML.
