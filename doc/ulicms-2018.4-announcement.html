UliCMS 2018.4 has many new features

The new UliCMS Release 2018.4 is the result of 3 months of development.

The release 2018.4 focuses on usability, privacy and performance features.

Major Features in this Release

  • Usability Improvements
  • Support for the legal requirements of the General Data Protection Regulation (GDPR)
  • Better Performance

Usability Improvements

Tables are now shown with the DataTable jQuery Plugin. Tabellen mit Datensätzen werden nun auf mehreren Seiten angezeigt (Pagination). Es kann festgelegt werden, wie viele Einträge je Seite angezeigt werden sollen. Des weiteren sind alle Tabellen nun durchsuchbar.

Die Formulare "Seite erstellen" und "Seite bearbeiten" sind nun übersichtlicher. Die Anzahl der Unterbereiche wurde reduziert, in dem diese vereinigt wurden. Selten genutzte Optionen wurden nach "Sonstiges" verschoben.

Support for GDPR Requirements

In den neuen Datenschutz-Einstellungen kann eine Checkbox zum Akzeptieren der Datenschutzerklärung hinzugefügt werden. Diese kann in Formulare und über eine Programmierschnittstelle in eigene Module integriert werden.

Das Recht auf Auskunft und das Recht auf Löschung der gespeicherten personenbezogenen Daten kann mit dem Modul personal_data_export umgesetzt werden.

Improved Performance

Die Performance wurde verbessert. Einige Datenbanktabellen wurden optimiert durch setzen von Indizes.

Javascript- und CSS-Codes werden nun besser reduziert. Außerdem werden nun unnötige Leerzeichen aus dem HTML entfernt.
