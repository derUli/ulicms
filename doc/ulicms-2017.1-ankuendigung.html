UliCMS 2017.1 ist da!

UliCMS 2017.1 ist das UliCMS Release für das erste Quartal 2017. UliCMS 2017.1 ermöglicht Webmastern, weiterhin ihre Webseiten zu betreiben.

UliCMS 2017.1 enthält eine Vielzahl neuer Features und Verbesserungen. Dazu gehören:

  • Viele neue und aktualisierte Pakete
  • Ein mehr an MVC angelehntes API
  • Verbessserungen der Benutzerfreundlichkeit
  • Neuer Inhaltstyp "Knotenpunkt"
  • Ein Erweitertes Berechtigungssystem
  • Ein neues Standard Theme, basierend auf bootstrap.css
  • Das neue Portal zum Download von Erweiterungen UliCMS eXtend

Hinweise zum Upgrade

Bevor Sie ein Upgrade auf UliCMS 2017.1 durchführen, stellen Sie sicher, dass bei Ihrem Benutzer die Option "Ist Admin" aktiv ist.

Nach dem Upgrade müssen Sie die neuen Berechtigungen vergeben. Wenn Sie zuvor Mobile_Detect genutzt haben, müssen Sie das Modul Mobile_Detect aus UliCMS eXtend installieren. Außerdem sollten Sie mit update_manager auf Aktualisierungen prüfen und diese installieren.
