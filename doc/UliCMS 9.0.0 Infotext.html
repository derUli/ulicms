Mit UliCMS 9.0 erscheint die nächste Generation des Open Source Content Management Systems. Diese wurde um praktische HTML5 Features für Multimedia Integration, sowie Sicherheitsfeatures, für eine höhere Systemsicherheit erweitert und trägt somit dazu bei, dass Sie als UliCMS Nutzer mit den neuesten Entwicklungen im World Wide Web mithalten können.

Multimedia mit HTML5

Videos und Musik Player in eine Website einbinden?
Dafür wurde früher oftmals der Flash Player von Adobe genutzt. Doch nun gibt es HTML5 und viele Browser unterstützen die Wiedergabe von Videofilmen und Audiodateien auch ohne zusätzliche Plugins.

UliCMS 9.0 integriert einen HTML5 Video und Audio Manager und macht es somit noch einfacher, zeitgemäßes Multimedia zur Aufwertung Ihrer Internetseite einzubinden. Für Nutzer älterer Browser ist auch gesorgt - Wenn der Browser kein HTML5 unterstützt, bekommt der User die Mediendatei einfach als Download angeboten und kann diese dann mit seinem bevorzugten Mediaplayer wiedergeben.

Sicherheit

UliCMS nutzt nun einen Token Check und dadurch resistent gegen sogenannte Cross-Site-Request-Forgery Angriffe

Patch Management

Patches können nun direkt im Backend installiert werden. Sobald neue Patches verfügbar sind, wird der Administrator im Backend des CMS darüber informiert.
Somit ist sichergestellt, dass die neusten Sicherheitsfixes schnell installiert werden können.
