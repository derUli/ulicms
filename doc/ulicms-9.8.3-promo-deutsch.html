UliCMS 9.8.3 ist da!

Mit UliCMS 9.8.3 bekommen Sie das neueste Release des deutschen Content-Management-Systems. Sie können Ihre bestehenden Website weiterhin mit UliCMS 9 betreiben, oder neue Projekte damit beginnen.

UliCMS 9.8.3 ist ein wichtiges Update, welches zahlreiche neue Features für Ihren Erfolg im World Wide Web enthält.

Diese Version von UliCMS enthält alle Features die in UliCMS 9.8.2 enthalten waren, sowie die folgenden neuen Features.

  • Klassen Banner und Banners hinzugefügt
  • Shell-Script sinfo.php hinzugefügt
  • Shell-Script sremove.php hinzugefügt
  • Shell-Script version.php hinzugefügt
  • Funktion containsModule() zur Klasse Page hinzugefügt
  • Funktion removeDir() zur Klasse Path hinzugefügt
  • Funktionen zur Klasse ContentFactory hinzugefügt
  • Klasse Request hinzugefügt
  • Klasse Cache hinzugefügt
  • Funktionen zur Template Klasse hinzugefügt
  • Aktualisierte Komponenten
  • Neue Hooks hinzugefügt
