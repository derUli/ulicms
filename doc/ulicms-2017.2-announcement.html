UliCMS 2017.2 is here

UliCMS 2017.2 is the second quarter of 2017 release of the german solutions based content management system. UliCMS enables webmasters to manage small to medium sized web portals.

Release 2017.2 contains a lot of improvements including:

  • New responsive backend design
  • Package and Patch Management now uses https://
  • Function ModuleHelper::getFirstPageWithModule() added
  • DBMigrator added

Upgrade Notes

An upgrade can be performed only from release 2017.1 as base.

Before you upgrade your existing installation to UliCMS 2017.2 you need to make sure that your user has "is admin" option checked.

After upgrading please assign the new permissions to groups. If you use Mobile_Detect in your project you have to install Mobile_Detect module from UliCMS eXtend. You have also to check for updates using update_manager.
