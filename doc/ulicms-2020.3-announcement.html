UliCMS 2020.3 reenables the page filters, RESPONSIVE filemanager and improves usability and performace

I hope you are keeping yourself in good health in these hard times. More than one month later than originally scheduled I present the new UliCMS version 2020.3 which is the main release for the third quarter of 2020.

The main features of this new release are the reintroduced page filters and the new RESPONSIVE filemanager that removes the dead kcfinder component for media file upload and management. Other features include better usability and performance improvements.

Page Filters

Mit dem Release von UliCMS 2020.2 wurden die Dropdown-Filter für Seiten wieder hinzugefügt. Jedoch war das Feature noch nicht ausreichend getestet und war daher deaktiviert. Mit UliCMS 2020.3 sind die Seitenfilter nun wieder aktiviert

RESPONSIVE filemanager

RESPONSIVE filemanager ist eine Dateimanager-Komponente, welche den in die Jahre gekommenen kcfinder zum Upload und der Auswahl von Dateien ersetzt.

Other Improvements

Shortcodes werden jetzt automatisch in die Zwischenablage kopiert. Die doppelte Vergabe von Slugs innerhalb einer Sprache wird nun verhindert. Neue Robo Tasks wurden hinzugefügt. Außerdem wurden diverse Usability und Performance Verbesserungen vorgenommen.
