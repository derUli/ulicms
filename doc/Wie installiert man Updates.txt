Module und Themes:
Es wird nicht automatisch auf verfügbare Updates der installierten Pakete geprüft.
Im Falle eines kritischen Updates, wie z.B. der Behebung einer Sicherheitslücke, informiere ich Sie auf dem UliCMS Portal darüber.
Sie sollten deshalb in regelmäßigen Abständen danach schauen.
Installierte Pakete können aktualisiert werden, in dem die neue Version über die bestehende drüber installiert wird.
Dies kann entweder aus einer Paketquelle oder per Upload eines UliCMS Pakets geschehen.
Bestehende Dateien werden dabei ersetzt, so dass manuelle Anpassungen am Code verloren gehen.

UliCMS Kernsystem:
UliCMS prüft seit Version 7.0 ob neue Updates verfügbar sind, und informiert Sie gegebenfalls im Dashboard darüber.
Upgrade-Pakete werden gegebenfalls als Download angeboten.
Diese müssen entpackt und per FTP hochgeladen werden.
Bestehende Dateien müssen dabei ersetzt.
Sofern vorhanden, müssen Sie der Datei update.php auf dem FTP-Server volle Berechtigungen geben.
Anschließend müssen Sie diese in Ihrem Browser aufrufen (z.B. http://domain.de/update.php). 
Das Script nimmt die notwendige Änderungen an der Datenbank vor und
löscht sich anschließend selbst.
