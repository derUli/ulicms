UliCMS 2020.1 with avatar upload, improved performance and support to set robots metatag by page

UliCMS 2020.1 offers a large amount of new features for designing your website project.

Performance

The page list in backend is now paginated. That means that only the current visible datasets are loaded into the memory. That improves the backend performance in projects with more than a few hundred pages drastically.

Generated PDF documents will now also be cached.

Avatar

User avatars based on the initial letters of first and lastname are generated.

Users can upload their own avatar photos which are replacing his initial letter avatar.

Robots Metatag

The robots meta tag can no be set individually for any page.
