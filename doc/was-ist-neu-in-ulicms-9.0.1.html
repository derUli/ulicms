Was ist neu in UliCMS 9.0.1?

UliCMS 9.0.1 ist eine neue Version des deutschen Content Management Systems, die einige gute neue Funktionen enthält, die Ihr Leben im Web vereinfachen werden.

Warum wurde diese Version von UliCMS veröffentlicht?

Diese Version von UliCMS wurde veröffentlicht, um den Anforderungen einer größeren Zielgruppe zu genügen. Erfahren Sie hier, welche neuen Features und Verbesserungen die neue Version von UliCMS bietet

Versionsverwaltung

UliCMS hat nun eine Versionsverwaltung für Seiteninhalte. Eine Versionsverwaltung sorgt dafür, dass beim Speichern eines Dokuments die alte Version erhalten bleibt. Wenn Sie nun ein Fehler machen wie z.B. aus Versehen einen Absatz zu löschen, können Sie die Seite auf eine ältere Version zurücksetzen. Es gibt außerdem die Möglichkeit, zwei Versionen einer Seite zu vergleichen und die Unterschiede zu visualisieren.

Die Versionsverwaltung von UliCMS 9.0.1 macht Ihr Leben im World Wide Web ein Stück sicherer.

Favicon Upload

Sie können nun ein sogenanntes Favicon hochladen. Favicons sind kleine Symbole, die im Browser angezeigt werden. Diese sorgen für einen größeren Wiedererkennungswert Ihrer Internetseite. Favicons können über eine grafische Oberfläche im .ico Dateiformat hochgeladen werden. Dabei wird die Datei per Meta-Tag automatisch von UliCMS in den HTML-Code eingebunden. Es sind keine Template Anpassungen dafür erforderlich.

Seiten-Zugriff beschränken

Es ist nun möglich, den Zugriff auf bestimmte Seiten so zu beschränken, dass diese nur auf mobilen Geräten oder auf Desktop PCs angezeigt werden. Dies ermöglicht die Reduzierung des Navigationsmenüs und das Ausblenden von Inhalten, die für den User auf der jeweiligen Plattform nicht wichtig sind.

Wenn Sie beispielsweise einen Service für Smartphone User anbieten, macht es eventuell Sinn, diesen Service auch nur auf den Geräten der Zielgruppe anzuzeigen.

Verbesserte Performance

Wir haben daran gearbeitet, die Performance von UliCMS zu verbessern. Darunter fallen optimierte Datenbankabfragen, die Kombinierung von Javascript-Dateien, sowie die verstärkte Nutzung vom Browser Cache.

Super einfaches Upgrade

Von UliCMS 9.0.0 zu upgraden ist einfach. Ersetzen Sie einfach die Dateien der UliCMS Installation mit denen aus dem "ulicms" Ordner aus dem Upgrade-Paket. Rufen Sie dann die Datei update.php (z.B. www.beispiel.de/update.php) im Browser auf, damit es die Datenbank wie benötigt anpasst. Es sollte keinerlei Kompatiblitätsprobleme mit Modulen oder Templates geben.
