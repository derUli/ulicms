UliCMS 2019.2 mit verbessertem User Interface

Look and Feel

Alle Buttons im Backend sind nun mit Icons versehen.

Die "Pakete" Seite im Backend wurde komplett neu gestaltet. Die neugestaltete Paketverwaltung enthält alle Features wie das alte User Interface. Zusätzlich ist es nun möglich Module im Backend zu aktivieren und zu deaktivieren.

Zusätzlich können Module nun aktiviert und deaktiviert werden. Dies war vorher nur mit dem Modul cleanup oder direkt in der Datenbank möglich. Es ist in diesem Release möglich, auf die alte Ansicht der "Pakete" Seite zu wechseln. Die alte Ansicht wird in einem zukünftigen Release entfernt werden.

Neue Sicherheits- und Antispam-Features

Die Optionen "XSS-Schutz" und "Erlauben diese Webseite in andere Sites einzubinden" zur Steuerung von sicherheitsrelevanten HTTP-Headern wurden hinzugefügt.

Entwickler Features

Ein SCSS Compiler wurde hinzugefügt und in die Stylesheet Queue integriert.
