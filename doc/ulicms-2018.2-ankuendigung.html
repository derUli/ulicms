UliCMS 2018.2 veröffentlicht

UliCMS Release 2018.2 ist eine freie Open Source Content Management Software aus Braunschweig / Deutschland und der Nachfolger von UliCMS 2018.2.

Neue Features in diesem Release

  • Benutzerfreundlichkeit und Design der grafischen Oberfläche verbessert
  • Verbesserte Sicherheit (Timestamp Antispam Methode, Integritätsprüfung für Downloads)
  • Features für Webentwickler
