UliCMS 2021.3 latest update for the third quarter of 2021

UI Improvements

Die Benutzerschnittstelle des Kommentarsystems wurde überarbeitet. Es wurde eine Animation zum Hamburger Menu Icon im Backend hinzugefügt.

Packages Outdated

Alle npm und composer Pakete wurden aktualisiert. CKEditor wurde auf Version 4.16.2 aktualisiert.

Bugfixes

Es wurden zwei SQL-Fehler korrigiert, die auftreten, wenn in der Konfigurationsdatei $db_strict_mode = true ist.
