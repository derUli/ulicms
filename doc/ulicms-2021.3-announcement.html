UliCMS 2021.3 latest update for the third quarter of 2021

UI Improvements

The user interface of the comment system got a revise. An animation was added to the hamburger menu.

Packages Upgraded

All npm und composer packages were updated. CKEditor is now at version 4.16.2.

Bugfixes

Two SQL errors happening while $db_strict_mode is true were fixed.. Es wurden zwei SQL-Fehler korrigiert, die auftreten, wenn in der Konfigurationsdatei $db_strict_mode = true ist.
