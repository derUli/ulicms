===== Neues in UliCMS 7.2.1 ====
+ CKEditor auf Version 4.4.5 aktualisiert
+ Editor wächst mit Größe
+ Nun auch Einbindung von Bannern über HTML-Code möglich
+ Datenbankstruktur verbessert
  - Tabelle 'admins' in 'users' umbenannt
  - Nutzung von Fremdschlüsseln
+ Es können Links zu lokalen Seiten eingefügt werden
+ diverse API Funktionen hinzugefügt
+ Optionen hinzugefügt
  - Gebe "generator" Meta-Tag aus
  - Design Optionen aktiviert
+ Sonstige Einstellungen nun in Accordion-Layout
+ Menüpunkte können optional als Grafik angezeigt werden
+ Für die Erstellung eines Drop-Down Menüs können nun Anker-URLs als externe URL angegeben werden.
  - Zuvor führte dies zu einer endlosn HTTP-Weiterleitung.
+ Neues UliCMS Logo
+ Mehrsprachigkeit verbessert
  - sprachspezifischer homepage_title möglich
  - sprachspezifisches motto möglich
  - Meta Beschreibung & Keywords je Sprache
  - Startseitenauswahl in den Einstellungen nun nach Sprache gefiltert
  - Es können Seiten mit dem gleichen Systemnamen in verschiedenen Sprachen existieren.
  - Backend kann nun per Sprachdatei lokalisiert werden
