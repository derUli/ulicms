===== Neues in UliCMS 9.0.2 ====
+ CKEditor auf Version 4.5.3 aktualisiert (Behebt Kompatiblitätsprobleme mit Microsofts neuem Edge Webbrowser)
+ Auswahl für Schriftart und Schriftgröße im CKEditor entfernt
+ Folgende Plugins in den CKEditor integriert
  - Auto Embed
  - CodeMirror
  - Media Embed
  - Word Count & Char Count Plugin
+ Skin "Office 2013" für CKEditor hinzugefügt
+ Option "IP-Adressen nach 48 Stunden löschen" hinzugefügt
+ Shell-Script delete_log_ips.php hinzugefügt
+ jQuery auf Version 1.11.3 aktualisiert
+ Referrer Spalte zu Protokoll hinzugefügt
+ Module können nun Metadaten haben (metadata.json)
+ Versionsnummern von Modulen und Designs können nun angezeigt werden
+ Mobile_Detect Klasse auf Version 2.8.15 aktualisiert
+ Bugfix: Werte in der Spalte "Menü" in der Seitenliste werden nun übersetzt
+ IP Whitelist hinzugefügt. In der Datei cms-config.php kann nun ein Array mit IP-Adressen als Variable ip_whitelist definiert werden.
  Sofern dieser Wert gesetzt ist, wird der Zugriff zum UliCMS Backend auf bestimmte
  IP-Adressen beschränkt. Die Nutzung dieser Option wird nur bei Internet-Zugängen mit einer 
  statischen IP empfohlen, wie dies z.B. in Firmennetzwerken oft der Fall ist.
+ Checkboxes und Radiobuttons für mobile Geräte optimiert
+ CSS-Klasse für Page-ID zu Menülinks hinzugefügt
+ Löschen von Seiten nun per AJAX möglich
+ Löschen von Bannern nun per AJAX möglich
+ Löschen von Kategorien nun per AJAX möglich
+ Löschen von Benutzern nun per AJAX möglich
+ Löschen von Gruppen nun per AJAX möglich
+ Löschen von Audio Dateien nun per AJAX möglich
+ Löschen von Videos nun per AJAX möglich
+ Löschen von Modulen nun per AJAX möglich
+ Löschen von Themes nun per AJAX möglich
+ Löschen von Einstellungen im Expertenmodus nun per AJAX möglich
+ Löschen von Sprachen nun per AJAX möglich
+ Bugfix: Berechtigung "remove_packages"  wird nun korrekt ausgewertet
+ API-Funktion is_ajax_request() hinzugefügt
+ Filter für Seitentitel wieder hinzugefügt
+ page-id CSS Klasse zum Body hinzugefügt
+ title_tag Filter hinzugefügt
+ Es können nun Open Graph Meta-Tags festgelegt werden.
+ Bugfix: API-Funktion all_combined_html() korrigiert. Der Aufruf dieser Methode führte zuvor zu einer Endlosschleife
+ Liste der bereits installierten Patches zur Paketverwaltung hinzugefügt
+ Favicon Generator akzeptiert nun auch andere Formate als .ico. Die Grafikdatei wird automatisch in das Favicon Format konvertiert
+ "modules" und "templates" Ordner nach "content" verschoben
+ Mobile CSS nun auch per Browser bearbeitbar
+ Individuelle Kontaktformulare können nun erstellt werden
+ Many Smartphones make phone numbers auto clickable. This is now disabled by default.
+ tel Tags are now supported. example: [tel]+49123456789[/tel]
+ skype Tags are now supported. example: [skype]MySkypeName[/skype]
+ Copyright-Jahr von UliCMS nun an zentraler Stelle
+ Bugfix: Beim Loginformular wurde immer die Sprachdatei der Standardsprache geladen. 
Nun wird die Sprache, die für die Domain, über die das Backend aufgerufen wird, definiert ist, geladend.
