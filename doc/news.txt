===== Neues in UliCMS 7.2.1 Service Pack 1 ====
+ Alle bisher erschienen Patches für UliCMS 7.2.1 enthalten
+ Zugriffsbeschränkungen ("Sichtbar für") für Seiten korrigiert und erweitert
+ no_cache() API Funktion hinzugefügt
+ Kein Caching bei eingeloggten Nutzern.
+ Es wird auch der Alternate PHP Cache (APC) geleert, sofern das 
entsprechende PHP-Modul installiert ist.
+ CKEditor auf Version 4.4.6 aktualisiert
+ Falsche Übersetzung von "Header Hintergrundfarbe" in den Designeinstellungen korrigiert
+ Seiten filtern nach Menü und übergeordneter Seite nun möglich
+ Theme nun pro Seite festlegbar
+ Kategorien haben nun auch eine Beschreibung
+ HTML5 required Attribut hinzugefügt

﻿===== Neues in UliCMS 7.2.1 ====
+ CKEditor auf Version 4.4.5 aktualisiert
+ Editor wächst mit Größe
+ Nun auch Einbindung von Bannern über HTML-Code möglich
+ Datenbankstruktur verbessert
  - Tabelle 'admins' in 'users' umbenannt
  - Nutzung von Fremdschlüsseln
+ Es können Links zu lokalen Seiten eingefügt werden
+ diverse API Funktionen hinzugefügt
+ Optionen hinzugefügt
  - Gebe "generator" Meta-Tag aus
  - Design Optionen aktiviert
+ Sonstige Einstellungen nun in Accordion-Layout
+ Menüpunkte können optional als Grafik angezeigt werden
+ Für die Erstellung eines Drop-Down Menüs können nun Anker-URLs als externe URL angegeben werden.
  - Zuvor führte dies zu einer endlosen HTTP-Weiterleitung.
+ Neues UliCMS Logo
+ Mehrsprachigkeit verbessert
  - sprachspezifischer homepage_title möglich
  - sprachspezifisches motto möglich
  - Meta Beschreibung & Keywords je Sprache
  - Startseitenauswahl in den Einstellungen nun nach Sprache gefiltert
  - Es können Seiten mit dem gleichen Systemnamen in verschiedenen Sprachen existieren.
  - Backend kann nun per Sprachdatei lokalisiert werden
  - Backend in Englisch übersetzt
  - Sprache zu Werbebanner System hinzugefügt
+ Cache in der Standardkonfiguration deaktiviert
+ Benutzerdefinierte Werte, diese müssen vom Benutzer als JSON String eingegeben werden.
  + Funktion get_custom_data zum Abfragen dieser Werte.
  + Funktion set_custom_data zum setzen eines Custom Data Werts
  + Funktion delete_custom_data zum löschen von Custom Data Werten
  + Funktion delete_custom_data zum löschen von Custom Data Werten
+ Funktion db_prepared_query für Prepared SQL Statements zur Datenbank API hinzugefügt
+ Neuen Font Stack "Segoe" zur Auswahl der Schriftart in den Designeinstellungen hinzugefügt
+ JSON Import entfernt, da diese Funktionalität scheinbar defekt ist
+ Advertising-Klausel zur Lizenz hinzugefügt. Bei Werbung für Produkte auf Basis von UliCMS muss künftig ein Hinweis, dass das Produkt Software von UliCMS / Ulrich Schmidt enthält erfolgen.
+ Neues Standard Design "cleanblue"
