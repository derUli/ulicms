+ Zugriffsbeschränkungen ("Sichtbar für") für Seiten korrigiert und erweitert
+ no_cache() API Funktion hinzugefügt
+ Kein Caching bei eingeloggten Nutzern.
+ Es wird auch der Alternate PHP Cache (APC) geleert, sofern das 
entsprechende PHP-Modul installiert ist.

+ Falsche Übersetzung von "Header Hintergrundfarbe" in den Designeinstellungen korrigiert
+ Seiten filtern nach Menü und übergeordneter Seite nun möglich
+ Theme nun pro Seite festlegbar
+ Kategorien haben nun auch eine Beschreibung
+ HTML5 required Attribut hinzugefügt
+ file_get_contents_wrapper cacht Downloads
+ Noembed Module
