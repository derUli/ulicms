===== Neues in UliCMS 9.0.0 ====
+ HTML 5 Video hinzugefügt
+ HTML 5 Audio hinzugefügt
+ Flash Dateiverwaltung entfernt
+ Online Updates: Patches können nun direkt aus dem UliCMS Backend installiert werden. Dies setzt vorraus, dass die Berechtigungen so gesetzt sind, dass alle Dateien und Ordner der UliCMS Installation durch PHP beschreibbar sind.
+ integriertes jQuery auf Version 1.11.1 aktualisiert
+ Anti CSRF Tokens integriert
+ jQuery nun auch im Frontend automatisch integriert. Sofern Sie jQuery als Modul installiert haben, müssen Sie dieses Modul deinstallieren, da jQuery ansonsten doppelt eingebunden wird.
+ Scroll Up Button rechts unten hinzugefügt. Nur auf Desktop PCs.
+ Installer versucht, die Datenbank anzulegen, falls diese noch nicht existiert
+ no_auto_cron Option hinzugefügt. Wenn diese gesetzt ist, dann wird die cron.php nicht automatisch ausgeführt.
+ style.css wird nun automatisch eingebunden wenn vorhanden.
+ CSS Klasse li.menu_active_list_item hinzugefügt
+ "log" Datenbanktabelle für Logging von HTTP Anfragen hinzugefügt.
  - IP Adressen werden nicht geloggt, dies kann jedoch durch setzen von log_ip geändert werden, verstößt jedoch gegen das deutsche Datenschutzgesetz
+ API Befehle get_request_method, get_request_uri, get_http_host, log_request, rand_string, get_user_id hinzugefügt
+ Datenbankspalte "type" zu cotent Tabelle hinzugefügt.
+ Textfeld für additional_menus zu Designeinstellungen hinzugefügt
+ HTML Editor nun je User festlegbar
  - Auswahlmöglichkeiten sind 
    (X) CKEditor (WYSIWYG, für Anfänger)
    (X) CodeMirror (HTML Code mit farblicher Hervorhebung, für HTML-Experten)
+ Registrierungsformular übersetzt
+ Funktion "Passwort zurücksetzen" hinzugefügt
