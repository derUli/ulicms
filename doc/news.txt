===== Neues in UliCMS 9.0.2 ====
+ CKEditor auf Version 4.5.1 aktualisiert
+ Skin "Office 2013" für CKEditor hinzugefügt
+ Option "IP-Adressen nach 48 Stunden löschen" hinzugefügt
+ Shell-Script delete_log_ips.php hinzugefügt
+ jQuery auf Version 1.11.3 aktualisiert
+ Referrer Spalte zu Protokoll hinzugefügt
+ Module können nun Metadaten haben (metadata.json)
+ Versionsnummern von Modulen und Designs können nun angezeigt werden
+ Mobile_Detect Klasse auf Version 2.8.15 aktualisiert
+ Bugfix: Werte in der Spalte "Menü" in der Seitenliste werden nun übersetzt
+ IP Whitelist hinzugefügt. In der Datei cms-config.php kann nun ein Array mit IP-Adressen als Variable ip_whitelist definiert werden.
  Sofern dieser Wert gesetzt ist, wird der Zugriff zum UliCMS Backend auf bestimmte
  IP-Adressen beschränkt. Die Nutzung dieser Option wird nur bei Internet-Zugängen mit einer 
  statischen IP empfohlen, wie dies z.B. in Firmennetzwerken oft der Fall ist.
