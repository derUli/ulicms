===== Neues in UliCMS 8.0.1 ====
+ Bedienung von UliCMS auf Mobile Devices wie Smartphones und Tablets optimiert
+ API Methoden register_action und remove_action hinzugefügt
  Damit kann man eigene Actions zum Backend hinzugefügen
+ Listen im Backend werden über jQuery Tablesorter sortiert, statt per SQL -> dadurch kein Page Reload mehr -> Sortierfunktion schneller
+ Sprachdateien werden auch im Frontend geladen - Dadurch Lokalisierung von Modulen möglich.
+ Fehlende Übersetzungen hinzugefügt (Grundeinstellungen, Fehlerseiten, etc.)
+ Es wird nun eine Fehlermeldung bei einem fehlgeschlagenen Login angezeigt.
+ Filtern nach "Aktiv" Status nun in der Seitenliste möglich
+ Newsfeed von www.ulicms.de eingebunden
+ "Cache leeren" nun direkt im "Einstellungen" Menü
+ Aktualisierung von mPDF auf mPDF 6.0
+ API Methoden get_translation, translate und translation hinzugefügt
+ wordcount Plugin zu CKEditor hinzugefügt
+ API Funktion get_site_protocol hinzugefügt
+ CodeMirror auf Version 4.12.0 aktualisiert
