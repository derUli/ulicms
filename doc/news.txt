===== Neues in UliCMS 9.8.4 =====

Neue Features
+ Inhaltstyp "Liste" hinzugefügt
+ Inhaltstyp "Link" hinzugefügt
+ Inhaltstyp "Modul" hinzugefügt
+ Inhaltstyp "Audio" hinzugefügt
+ Inhaltstyp "Video" hinzugefügt
+ Tab "Position der Beschreibung" hinzugefügt
+ Klasse ModuleHelper hinzugefügt
+ post-install Scripts können nun auch manuell ausgeführt werden
+ Rudimentäre Freigabe-Workflows sind nun möglich
+ Eigentümer einer Seite kann nun geändert werden
+ Benutzerfreundliches des Installationsassistenten verbessert

Änderungen
# Feld "Position" ist jetzt ein HTML5 Number Feld

Entfernte Features
- Templates können nicht mehr online im Browser bearbeitet werden
