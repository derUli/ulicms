UliCMS 2019.1 mit vielen neuen Features

UliCMS 2019.1 ist das erste UliCMS Release im Jahr 2019.
Dieses Release legt den Hauptfokus auf Community Features, Antispam Features und auf Verbesserungen der Usability.

Major Features in diesem Release

  • Kommentarfunktion
  • Zusätzliche Optionen zur Bekämpfung von Spam
  • Mehrere Gruppen einem Benutzer zuweisen
  • "Löschen" Button in der "Seite bearbeiten" Ansicht
