UliCMS 2019.4 supports auto embeddable media, hierarchical and has a new standard page

UliCMS 2019.4 offers a large amount of new features for designing your website project.

Auto embed external media ressourcexs

URLs von externen Medien werden bei Aufruf einer Seite automatisch durch eingebettete Media Player ersetzt. Es werden über 150 Dienste unterstützt. Unter den unterstützten Diensten befinden sich unter anderem die Folgenden:

  • YouTube
  • Dailymotion
  • MyVideo
  • Vimeo
  • Ustream

hierarchical page mangement

Die Seitenliste ist nun hierarchisch aufgebaut, näheres dazu im unten verlinkten Dokument "Neue Features in UliCMS 2019.4".

A new single page theme

UliCMS 2019.4 enthält als zusätzliche Gestaltungsoption das neue Single-Page Theme 2020. Wenn Sie kommerzielle Projekte auf Basis dieses Themes umsetzen möchten, lesen Sie bitt den Hinweis zur Lizenzierung von fullPage.js.
