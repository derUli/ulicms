UliCMS 2019.4 supports auto embeddable media, hierarchical and has a new standard page

UliCMS 2019.4 offers a large amount of new features for designing your website project.

UliCMS 2019.4 has again compatiblity

Support for older MySQL / MariaDB versions

UliCMS 2019.4 gained again compatiblity with MySQL 5.5.3 and similiar MariaDB versions, while the previous UliCMS release 2019.3 required at least MySQL 5.6.

Auto embed external media ressourcexs

URLs to external media ressources are now automatically replaced with embedded players. More than 150 different services are supported.

This and more media services are supported:

  • YouTube
  • Dailymotion
  • MyVideo
  • Vimeo
  • Ustream

hierarchical page mangement

The pages list is now managed as a hierarchical tree. You can find more Information about that in the document "Neue Features in UliCMS 2019.4" (german only).

A new single page theme

UliCMS 2019.4 features the new single page theme 2020 as an additional design option. If you want to use this theme in a commercial project instead of using it under the conditions of the GNU Public License then you have to purchase a license key for fullpage.js
