UliCMS 2018.2 is now available

UliCMS is the current major release of the german content-management-solution from germany. The developer released this new version to support users with new features and bugfixes.

UliCMS 2018.2 containts a wide-range of new features and bugfixes:

  • User Interface Usability and Design Improvements
  • Improved Security (Timestamp Antispam Method, Checksum-Validation for Downloads)
  • New Developer Features
