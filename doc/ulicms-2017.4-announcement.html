UliCMS 2017.4 is now available

UliCMS is the current major release of the german content-management-solution from germany. The developer released this new version to support users with new features and bugfixes.

UliCMS 2017.4 containts a wide-range of new features and bugfixes:

  • New content type "Snippet"
  • Assign languages to groups
  • New security features
  • Enhancements of UliCMS framework
  • Emoji Support
